# UI Library Documentation

## Overview

This UI library provides a set of reusable components that can be used to build user interfaces efficiently and consistently.

## Components

- **Buttons**: Customizable buttons with various styles and sizes.
- **Forms**: Accessible form components for user input.
- **Modals**: Responsive modals for displaying additional content without navigating away from the current page.
- **Notifications**: System for displaying notifications to users, such as success messages or alerts.

## Usage

To use the components, import them into your project as follows:

```javascript
import { Button, Modal } from 'your-ui-library';
```
